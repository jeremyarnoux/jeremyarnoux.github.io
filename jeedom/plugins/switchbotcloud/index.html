SwitchBot

Ce plugin permet de synchroniser les produits de la marque Switchbot qui sont configurés dans leur application dédiée. Lorsque le plugin est activé et configuré, vous retrouverez tous les matériels dans votre Jeedom. Vous pourrez ainsi lancer des commandes et/ou récupérer des informations via des scénarions ou autres. ATTENTION: il est impératif d'avoir accès à votre Jeedom en https pour que les informations remontent dans votre installation correctement.
